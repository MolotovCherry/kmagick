addImage

external fun addImage(addWand: MagickWand)
Content copied to clipboard

Adds a clone of the images from the second wand and inserts them into the first wand.

 

Use setLastIterator, to append new images into an existing wand, current image will be set to last image so later adds with also be appended to end of wand.

 

Use setFirstIterator to prepend new images into wand, any more images added will also be prepended before other images in the wand. However, the order of a list of new images will not change.

 

Otherwise, the new images will be inserted just after the current image, and any later image will also be added after this current image but before the previously added images. Caution is advised when multiple image adds are inserted into the middle of the wand image list.

Parameters

addWand

The wand that contains the image list to be added.
